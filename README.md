## Project Title: AD 311 Custom Alert Buttons in React
#### By: Amde Wubshet

### Breif Description: 
Developed a React application featuring a Toolbar component that houses multiple AlertButton components. This project is designed to strengthen your understanding of how to pass data via props (including the special children prop) and how to execute specific logic using event handlers.

### Commands 
After installing the repo, navigate into your project directory: *cd alert-button-toolbar*. Then start the development server: *npm run dev*.
